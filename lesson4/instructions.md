# מטרה
לשלב Flexbox, CSS Grid, טבלאות ותכונות CSS (Box Shadow, אנימציות) כדי לבנות רכיבי ממשק משתמש מהעולם האמיתי.
> *הרגישו חופשי לחזור לנקודות הזמן בווידאו שסופקו בכל משימה אם אתם נתקעים!*

---

## משימה 1: סרגל ניווט "דביק" (Sticky)

### נושאי ייחוס:
* [עוגנים וניווט (Anchors & Navigations) (5:34:25)](https://youtu.be/NeeSJ3ZJzEg?t=12865)
* [עקרונות Flexbox (4:17:08)](https://youtu.be/NeeSJ3ZJzEg?t=15428)
* [הצללת קופסה (Box Shadow) (6:20:05)](https://youtu.be/NeeSJ3ZJzEg?t=22805)
* [אנימציות (מעברי CSS - Transitions)](https://youtu.be/NeeSJ3ZJzEg?t=26425)

### הוראות:
1. צרו אלמנט `<nav>` שישמש כסרגל ניווט עליון לאתר.
2. השתמשו ב-**Flexbox** לפריסת הניווט. הלוגו צריך להיות בצד השמאלי הקיצוני, וקישורי הניווט (תגיות `<a>`) צריכים להיות מקובצים בצד הימני הקיצוני.
3. הוסיפו **Box Shadow** עדין לתחתית סרגל הניווט כדי שייראה כאילו הוא מרחף מעל תוכן העמוד.
4. **אנימציות:** הוסיפו מעבר CSS (transition) חלק לעוגנים. כאשר המשתמש מרחף (hover) מעל קישור, צבע הרקע שלו צריך להשתנות והוא אמור לזוז מעט כלפי מעלה (באמצעות `transform: translateY`).

## משימה 2: גלריית מוצרים רספונסיבית

### נושאי ייחוס:
* [גלישת Flexbox (Wrap) (4:27:37)](https://youtu.be/NeeSJ3ZJzEg?t=16057)
* [מידות ב-Flexbox (basis, shrink, grow) (4:36:51)](https://youtu.be/NeeSJ3ZJzEg?t=16611)
* [כלי מפתחים (DevTools) ל-Flexbox (4:50:06)](https://youtu.be/NeeSJ3ZJzEg?t=17406)

### הוראות:
1. צרו קונטיינר (container) מסוג `<div>` המכיל 5 עד 6 "כרטיסיות" מוצר (אלמנטי `<div>` פשוטים עם כותרת ותמונת ממלא-מקום).
2. הגדירו לקונטיינר `display: flex` וודאו שהכרטיסיות גולשות לשורה הבאה כשהמסך הופך לקטן מדי באמצעות `flex-wrap`.
3. השתמשו ב-`flex-basis` כדי להגדיר את הרוחב ההתחלתי של הכרטיסיות (למשל, `250px`).
4. השתמשו ב-`flex-grow` כך שאם יש מקום נוסף בשורה, הכרטיסיות הנותרות יתרחבו כדי למלא את השורה בצורה יפה. הגדירו `flex-shrink` כדי לוודא שהן לא שוברות את הפריסה במסכים קטנים מאוד.
5. **אתגר DevTools:** פתחו את כלי המפתחים בדפדפן שלכם (F12). לחצו על התג "flex" ליד הקונטיינר שלכם בבדיקת ה-HTML (Inspector). צלמו מסך של קווי כיסוי ה-Flexbox המראים כיצד הכרטיסיות שלכם מחולקות, ושמרו אותו להגשה יחד עם הקוד שלכם.

## משימה 3: טבלת מובילים (Leaderboard) אינטראקטיבית

### נושאי ייחוס:
* [טבלאות (Tables) (6:11:40)](https://youtu.be/NeeSJ3ZJzEg?t=22300)
* [הצללת קופסה (Box Shadow) (6:20:05)](https://youtu.be/NeeSJ3ZJzEg?t=22805)
* [אנימציות](https://youtu.be/NeeSJ3ZJzEg?t=26425)

### הוראות:
1. בנו `<table>` HTML מובנה המציג "טבלת משתמשים מובילים". כללו `<thead>` (דירוג, שם, ציון, סטטוס) ו-`<tbody>` עם לפחות 4 שורות של נתונים.
2. עצבו את הטבלה כך שתיראה מודרנית: בטלו את המרווח בין הגבולות (collapse), הוסיפו ריפוד (padding) לתאים (`<th>` ו-`<td>`), ויישרו את הטקסט כראוי.
3. עטפו את הטבלה כולה בקונטיינר `<div>` והחילו **Box Shadow** ברור ופינות מעוגלות (`border-radius`) כדי שזה ייראה כמו כרטיסיית דאשבורד.
4. **אנימציות:** הוסיפו אפקט ריחוף (hover) לשורות הטבלה (`tr:hover`). בעת ריחוף מעל שורה, צבע הרקע שלה צריך לעבור בצורה חלקה לצבע הדגשה בהיר, מה שיקל על קריאת הנתונים.

## משימה 4: רשת ווידג'טים לדאשבורד

### נושאי ייחוס:
* [עקרונות Grid (4:51:31)](https://youtu.be/NeeSJ3ZJzEg?t=17491)
* [מיקום פריטים ב-Grid (place-items) (5:05:51)](https://youtu.be/NeeSJ3ZJzEg?t=18351)

### הוראות:
1. צרו קונטיינר ראשי `<div>` והחילו עליו `display: grid`.
2. הגדירו פריסת רשת עם 3 עמודות ו-2 שורות באמצעות `grid-template-columns` ו-`grid-template-rows`. צרו מרווח (gap) בין פריטי הרשת.
3. צרו 6 אלמנטי `<div>` שישמשו כ"ווידג'טים" בתוך הרשת.
4. בואו נתמקד בווידג'ט הראשון. בתוך ווידג'ט זה, הוסיפו טקסט קצר (למשל, "אין הודעות חדשות").
5. החילו `display: grid` גם על הווידג'ט הספציפי הזה, והשתמשו בתכונה `place-items: center` כדי למרכז את הטקסט בצורה מושלמת בדיוק באמצע הווידג'ט, גם אופקית וגם אנכית.
